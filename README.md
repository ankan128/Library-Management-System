# 📚 Library Management System (Flask + MySQL)

A full-stack Library Management Web Application built using Flask, Python, and MySQL.  
This project allows users to register, login, search books, borrow books, return books, and manage a wishlist cart.

---

## 🚀 Features

- User Registration & Login System
- Search Books by Title/Author
- Issue (Borrow) Books
- Return Books
- Wishlist / Cart System
- Admin Dashboard (if implemented)
- MySQL Database Integration

---

## 🛠️ Tech Stack

- Python
- Flask
- MySQL
- HTML, CSS (Frontend)
- Jinja2 Templates
